Using Selective Media

  1. Double click first text field to select a folder containing media files that will be played between the time shown on the first time field and the time shown on the second time field.
  2. Double click second text field to select a folder containing media files that will be played between the time shown on the second time field and the time shown on the first time field, initially.
    The reverse will be the case (unless sequential is selected in #8) every other time media file is selected for playback, when it is selected from #3.
    The very first media file played may be from the folder from #4.
  3. Double click third text field to select a folder containing media files that will be played (every other time) between the time shown on the second time field and the time shown on the first time field.
  4. Doublic click fourth field to select a file that contains applications you want to run, instead of playing a media file.
    It is recommended to use a text file (.txt) for this purpose.
    You can cun multiple applications - put the full path to each application (e.g. C:\Windows\System32\Notepad.exe) in its own line, all of them will be run.
  5. Double click fifth text field to select a folder containing picture files that will be set (randomly) as desktop background (initially) between the time shown on the second time field and the time shown on the first time field.
  6. Whatever you type inside the sixth text field is going to be said (by the computer) initially if you set mode to one that is Sequential (#8).
  7. Select how fast you want media files to be played in succession - Ad is faster than Relaxed.
    If you choose any sequentially mode however, this rate is ignored, media files are played one after the other with little or no time interval.
  8. Select how you want media files to be played in succession - randomly or sequentially.

Additional Information

  • Right click the UI to edit or lock the text fields or to exit the program without saving any changes.
  • Double click the dialog window to show additional settings (you can change the time shown on the time fields).
  • Press close (x) to save all the above settings for every time Selective Media starts.
  • Right click the Icon in the System Tray to show this dialog window and to open Task Manager.
